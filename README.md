### 所需版本

- Hexo3.0及以上

### 安装

使用`yarn`或者`npm`命令

- yarn

```sh
yarn add hexo-theme-bubblegum
```

- npm

```sh
npm i hexo-theme-bubblegum
```

改变Hexo的`_config.yml`中的theme配置项

```yaml
# theme: landscape
theme: bubblegum
```

使用Hexo命令

```sh
hexo clean && hexo g
```

完成
