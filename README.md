### 所需版本

- Hexo3.0及以上

### 安装

在Hexo的`_config.yml`同级目录下，使用下面git命令

```sh
git clone https://github.com/MIMONATCH/hexo-theme-BubbleGum.git themes/bubblegum
```

改变Hexo的`_config.yml`中的theme配置项

```yaml
# theme: landscape
theme: bubblegum
```

使用Hexo命令

```sh
hexo clean && hexo g
```

完成
