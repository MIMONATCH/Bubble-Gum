### General

- hexo 3.0 or later

### Install

use `yarn` or `npm`

- yarn

```sh
yarn add hexo-theme-bubblegum
```

- npm

```sh
npm i hexo-theme-bubblegum
```

Change theme proprety in `_config.yml` 

```yaml
# theme: landscape
theme: bubblegum
```

Use Hexo command

```sh
hexo clean && hexo g
```
