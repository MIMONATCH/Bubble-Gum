### General

- hexo 3.0 or later

### Install

In the `root` directory

```sh
git clone https://github.com/MIMONATCH/hexo-theme-BubbleGum.git themes/bubblegum
```

Change theme proprety in `_config.yml` 

```yaml
# theme: landscape
theme: bubblegum
```

Use Hexo command

```sh
hexo clean && hexo g
```
